# Sales Performance Analysis 

## Project Overview
This project analyzes a superstore sales data to uncover trends, profitability issues, and business opportunities using Python and Power BI.
## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Jupyter Notebook

## Key Insights
- Technology is the most profitable category
- Furniture has high sales but low profit margin
- West region performs best
- South and Central regions require improvement
- Business depends heavily on one product

## Recommendations
- Review pricing, discounts, and operational costs in the Furniture category to improve profit margins
- Increase marketing and customer engagement efforts in the South and Central regions to improve sales performance
- Monitor and optimize discount strategies, especially in loss-making states such as Florida
- Develop targeted campaigns and tailored offerings to improve engagement with Home Office customers
- Reduce dependency on the Canon Image Copier by promoting and expanding other high-potential products
  
## Dashboard Features
- Sales trend analysis
- Regional performance
- Segment analysis
- Product analysis
- Profitability insights
